# Chronicle Weaver — Roadmap

## Current: v1.0.0 (Launch)
- [x] Multiple historical eras (Rome, Medieval, Renaissance, Wild West, Victorian)
- [x] AI-powered dynamic storytelling
- [x] Character development (Influence, Knowledge, Resources, Reputation)
- [x] Inventory and item system
- [x] NPC relationship management
- [x] Political, economic, and social world systems
- [x] Cross-platform (iOS, Android, Web)
- [x] Live at chronicleweaver.com
- [x] Firebase hosting and Firestore backend
- [x] CI/CD pipelines (multiple deploy workflows)
- [x] Enterprise standards (AGENTS.md, CHANGELOG.md, structured logging, release automation)

## v1.1.0 — Content Expansion
- [ ] New eras: Ancient Egypt, Feudal Japan, Roaring Twenties
- [ ] Expanded NPC dialogue trees with branching consequences
- [ ] Side quests and optional storylines (3-5 per era)
- [ ] Improved character customization (appearance, background, traits)
- [ ] Era-specific music and ambient sound

## v2.0.0 — Multiplayer & Social
- [ ] Cooperative story mode (2 players in same era)
- [ ] Player-created scenarios with publishing
- [ ] Community story sharing and rating
- [ ] Achievement system with era-specific badges
- [ ] Global and friends leaderboards
- [ ] Replay system for sharing story moments

## v3.0.0 — Advanced AI
- [ ] Voice-driven storytelling (speech-to-text input)
- [ ] Procedurally generated maps for each era
- [ ] Persistent world state across sessions and devices
- [ ] AI-generated illustrations for key scenes
- [ ] Dynamic difficulty adaptation based on player skill
